기존 홈페이지는 제품 라인업과 목적에 따른 추천 기능이 부족하고, 텍스트 중심의 설명으로 인해 기술에 대한 직관적인 체험이 어렵다는 한계가 있었습니다. 이번 리뉴얼은 '정보 탐색의 효율화'와 '브랜드 경험의 극대화'에 초점을 맞추었습니다. 복잡하게 분산되어 있던 정보를 직관적으로 재구성하고, 생동감 있는 라임 컬러 포인트와 인터랙티브 시각 요소를 도입하여 사용자가 온러닝의 혁신을 직접 체감할 수 있도록 개선했습니다.

This renewal goes beyond a simple design update — it was planned to significantly improve content accessibility and enhance the overall user experience. In particular, the renewal reflects an internal branding strategy that emphasizes the On Running brand over the corporate brand, with a strong focus on expanding touchpoints between users and On's original running content.

Project Goals

이번 온러닝 홈페이지 리뉴얼은 사용자 중심의 UX와 콘텐츠 중심의 트렌드를 결합하여 프리미엄 스포츠 브랜드의 새로운 가능성을 제시합니다. 변화하는 디지털 환경 속에서 온러닝의 경쟁력을 강화하고, 다양한 사용자 층의 니즈를 만족시키는 플랫폼으로 자리매김할 것입니다.

Core Value 1

다양한 디바이스에서 동일한 퍼포먼스를 제공하는

Core Value 2

직관적인 가이드로 탐색 경험을 향상시키는

Core Value 3

기술력을 생동감 있게 전달하는 시각적 인터랙션

Design Principle 1

Improvement & Flexible

러닝 목적, 사용 환경 기반 추천 인터랙티브 가이드로 탐색 과정 단순화.

Design Principle 2

Intuitive UI

주요 러닝화 모델을 한 화면에서 비교하는 직관적 퍼포먼스 UI 도입.

Design Principle 3

Brand Identity

자연의 생명력을 상징하는 네온 옐로우 라임 컬러 도입 및 스토리텔링 강화.

Design Elements

디자인 요소들은 온러닝의 정체성을 반영해 강렬하면서도 전문적인 분위기를 조성하며, 기술력을 강조해 활기찬 브랜드 경험을 제공합니다.

Color

  • BLACK #000000 RGB (0, 0, 0)
  • NEON YELLOW #D9EA1E RGB (217, 234, 30)
  • WHITE #FFFFFF RGB (255, 255, 255)

Typography

  • Noto Sans JP (KR) 가나다
  • Noto Sans JP (EN) Abc

MAIN

메인 페이지와 메가 네비게이션은 방대한 제품군(MEN, WOMEN, KIDS 등)을 한눈에 파악할 수 있도록 깔끔한 그리드와 인기 상품 썸네일을 활용하여 탐색 경험을 크게 개선했습니다.

COLLECTION

메인 페이지와 각 채널의 페이지는 동일한 기본 구조를 유지하며, 특별한 콜라보레이션을 돋보이게 하는 비주얼 레이아웃으로 컬렉션의 분위기를 극대화합니다.

PRODUCT COMPARISON

러닝화의 핵심 스펙, 무게, 퍼포먼스 등을 직관적인 표 형태로 한눈에 비교할 수 있어 사용자의 합리적인 구매 결정을 돕습니다.

DESIGN FEATURES

클라우드텍(CloudTec) 등 독보적인 기술력을 부드러운 스크롤 애니메이션과 디테일 컷으로 풀어내어 간결한 콘텐츠 배열로 사용자 편의성을 제공합니다.

RESPONSIVE

MOBILE
EXPERIENCE

다양한 디바이스 환경에서도 온러닝의 브랜드 가치를 일관되게 경험할 수 있도록 최적화된 반응형 인터페이스를 제공합니다.

01. MAIN

가장 역동적인 첫인상

모바일에서도 데스크탑의 웅장한 시각적 경험을 놓치지 않도록, 세로형 레이아웃에 최적화된 풀스크린 비주얼과 직관적인 퀵 네비게이션을 도입했습니다.

02. COLLECTION

직관적인 탐색과 필터링

한 손 조작이 용이하도록 하단 탭 스타일의 필터와 카드 리스트를 적용하여, 수많은 제품 라인업 사이에서도 원하는 러닝화를 빠르게 찾을 수 있습니다.

03. TECHNOLOGY

생동감 있는 기술력 전달

온러닝만의 '클라우드텍' 기술을 모바일에 최적화된 가벼운 애니메이션과 마이크로 인터랙션으로 풀어내어 브랜드의 신뢰도를 높였습니다.
